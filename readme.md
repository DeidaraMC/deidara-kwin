# Deidara KWin Shortcuts

A small collection of KWin scripts and shortcuts to improve my experience

## Install

```bash
cd ~/.local/share/kwin/scripts
git clone https://github.com/DeidaraMC/deidara-kwin.git
qdbus6 org.kde.KWin /KWin reconfigure
```

Then enable the script in:

```text
System Settings -> Window Management -> KWin Scripts
```

## Shortcuts

Set the shortcuts manually in:

```text
System Settings -> Keyboard -> Shortcuts
```

Current shortcut names:

```text
Switch screen left + move mouse
Switch screen right + move mouse
```
